We have got linters and static analyzers. However, the stated structure of the project plays an equally important role in keeping code clean. So, we need an automated tool for the linting file system of projects.

Installation

npm install --save-dev lint-fs

Download binary for the platform (Debian, Ubuntu, or Windows) on the release page.

Usage

  1. Create the lint-fs.yaml file in the project's root and specify the project's structure, using the power of ECMAScript regular expressions.
  2. Then just:
    • run node node_modules/lint-fs/npm/run
    • or execute the downloaded file from the root of your project
